You are a senior UX designer specialized in executive-level advisory firms serving healthcare boards and compliance officers.

I have exported each page of my AI Governance report as high-resolution PNG files.
These must be displayed exactly as they are (no redesign, no reformatting).

Your task is to design and structure a premium web-based document viewer component for my website.

This is NOT:
- A marketing carousel
- A SaaS product demo
- A playful flipbook
- A 3D book animation

This IS:
- A board-level document experience
- Calm, structured, institutional
- Designed for healthcare executives

--------------------------------------------------
CONTEXT
--------------------------------------------------

The document is an illustrative AI Governance Assessment.
It must feel:
• Serious
• Executive
• Structured
• Calm
• Trustworthy

--------------------------------------------------
COMPONENT REQUIREMENTS
--------------------------------------------------

1. Display one full report page at a time.
2. Maintain original document proportions.
3. Allow horizontal navigation (arrows + swipe).
4. Include subtle page indicator (e.g. “Pagina 1 / 7”).
5. Use fade + slide transition (max 300ms, ease-in-out).
6. No bounce effects.
7. No 3D page flip.
8. No drop shadows that feel playful.
9. No decorative UI elements.

--------------------------------------------------
LAYOUT
--------------------------------------------------

Above the viewer:

Title:
“Indicatieve bestuursrapportage”

Subtext:
“Voorbeeld van hoe een AI Governance Assessment wordt gepresenteerd.”

The subtext must clearly signal this is illustrative, not a client case.

The document viewer must:
• Be centered
• Have generous white space around it
• Max width 1100–1200px
• Slight subtle border (1px neutral gray)
• No heavy card styling

Below the viewer:

Subtle call-to-action:
“Wilt u weten hoe dit er voor uw organisatie uitziet?”
Button: “Plan een verkennend gesprek”

Button style:
• Solid dark
• No gradient
• No glow
• Calm hover state

--------------------------------------------------
INTERACTION DETAILS
--------------------------------------------------

Navigation arrows:
• Minimal
• Thin-line icons
• Positioned mid-height
• Low visual weight

Page indicator:
• Small
• Neutral color
• Bottom right corner

Mobile:
• Swipe enabled
• Arrows hidden
• Page indicator centered below

--------------------------------------------------
PERFORMANCE
--------------------------------------------------

Images must:
• Be optimized for web
• Lazy-loaded
• Maintain crisp resolution on retina screens
• Not distort proportions

--------------------------------------------------
PSYCHOLOGICAL GOAL
--------------------------------------------------

When a hospital board member views this, they should think:

“This is structured.”
“This feels serious.”
“This is not marketing.”

NOT:
“Nice sales trick.”

--------------------------------------------------
FINAL CHECK
--------------------------------------------------

Would this look credible if printed?
Does this feel institutional?
Is it restrained?
Does it avoid all startup aesthetics?

If not, simplify further.
